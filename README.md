  - Removed conflicting `public/sitemap.xml`

- **Problem**: Missing self-referencing canonical tags
- **Solution**: Implemented canonical URLs across all pages

- **Fix**: Removed `src/pages/sitemap.xml.ts` to resolve route conflict
